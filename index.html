Go Module Redirect for llib.dev

This site helps you use the custom import path llib.dev for your Go packages.

How to Use

When coding in Go, import your package like this:

import "llib.dev/your-package-name"

Example

To download a package called example, use this command:

go get llib.dev/example

What's Happening Behind the Scenes?

This site has special HTML files with code that tells the Go tool where to find the actual code for your package. So when you use go get, it knows where to look.
